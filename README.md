# 这是一个练习前端布局和后端登录验证的项目
## 前端
- 前端使用的语言为：HTML+CSS+JS
- 仿github的登陆模块主要文件结构为：
  - index.html：入口文件（主页）
  - sign_in_or_sign_up（文件夹）：用来存放登录相关页面
  - images（文件夹）：用来存放网站中用到的图片
  - scripts（文件夹）：用来存放脚本代码
  - styles（文件夹）：用来存放样式文件
